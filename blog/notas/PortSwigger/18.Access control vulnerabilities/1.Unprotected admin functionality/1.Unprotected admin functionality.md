En el sitio web encontraremos un robots.txt.

```c
/robots.txt
```

En este robots.txt encontraremos una ruta del administrador.

![20260219232243.png](20260219232243.png)

El cual al acceder al panel de administrador:

```c
/administrator-panel
```

Al ingresar podemos observar que el acceso esta permitido.
